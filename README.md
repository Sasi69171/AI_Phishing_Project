 AI Phishing Detection Web App

A web-based application that helps users identify phishing websites and suspicious URLs using a machine learning model trained on phishing and legitimate website datasets. The application aims to improve cybersecurity awareness and protect users from online threats.

 Features

* Phishing URL Detection
* Legitimate Website Identification
* Machine Learning-Based Prediction
* User-Friendly Interface
* Real-Time URL Analysis

 Technologies Used

* Python
* HTML
* CSS
* JavaScript
* Flask
* Machine Learning

Dataset

The model was trained using a dataset containing phishing and legitimate website URLs with various features used for classification.

 Model Training

* Data Collection and Preprocessing
* Feature Selection
* Model Training
* Model Evaluation and Testing
* Prediction Generation

Project Overview

The AI Phishing Detection Web App analyzes website URLs and predicts whether a website is phishing or legitimate. The system uses a trained machine learning model to classify URLs and provide users with security-related insights.
